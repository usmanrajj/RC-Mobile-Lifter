Components :- 
1: DC Motors + Wheels (4x)+ Chassis.
2: Motor Driver Shield L293D.
3: Arduino UNO.
4: Servo Motors.
5: Bluetooth Module (HC-05).
6: Jumper wires.
7: Battery 9-12V.
